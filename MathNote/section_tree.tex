\subsection{二叉树}

\begin{lemma}
深度为$d$的二叉树最多有$2^{d-1}$片树叶。
\label{tree_lemma_depth}
\end{lemma}

\begin{proof}
用数学归纳法证明。 

\begin{enumerate}
\item 当深度$d=1$时，二叉树只有一个根，最多存在一个树叶（就是根自己），所以基准情况为真。
\item 假设当深度为$d-1$时($d$是大于1的整数)，二叉树最有$2^{d-2}$片树叶。

那么对于深度为$d$的树$T$，其左右子树$T_1,T_2$，每一个的深度最多是$d-1$。根据假设，其左右子树最多有$2^{d-2}$个树叶。

$T$的树叶等于左右子树的树叶之和，其最多等于$2^{d-2} + 2^{d-2}  = 2^{d-1} $。


\end{enumerate}
\end{proof}



\begin{lemma}
具有$L$片树叶的二叉树深度至少是$ \lceil \log L \rceil$
\end{lemma}

\begin{proof}
由引理(\ref{tree_lemma_depth})可立即得出。
\end{proof}



\subsection{二叉查找树}

\emph{二叉查找树}(Binary Search Tree)是指一颗空树或者具有如下性质的二叉树：
\begin{enumerate}
\item 若任意节点的左子树不空，则左子树上所有节点的值均小于它的根节点的值;
\item 若任意节点的右子树不空，则右子树上所有节点的值均大于它的根节点的值;
\item 任意节点的左、右子树也分别为二叉查找树;
\item 没有键值相等的节点。
\end{enumerate}


二叉查找树相对于其他数据结构的优势在于查找、插入的时间复杂度较低，为$\bigo{\prt{\log N}}$。
